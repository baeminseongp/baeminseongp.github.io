object tracking and balancing robot

Introduction to Robotics Design course project.

We used an IMU sensor to maintain the horizontal position of the plate according to acceleration. We also created a robot that can follow objects without losing them by using four Mecanum wheels.

Key Features

  • Color tracking
  • Mecanum wheel kinematics for omni‐directional motion
  • PID‐based motor control
  • Horizontal direction maintenance control using IMU

Tech Stack

  • Arduino & C
  • Pixy camera
