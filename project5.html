4WD Mecanum Color‑Tracking Robot

Introduction to Robotics Design course project.

A four‐wheel mecanum robot that follows colored markers on the floor using a color‑detection algorithm and feedback control to navigate complex paths.

Key Features

  • Color segmentation and marker tracking
  • Mecanum wheel kinematics for omni‐directional motion
  • PID‐based motor control
  • ROS1 integration and pigpio encoder feedback

Tech Stack

  • ROS1 & Python
  • OpenCV (Color detection)
  • Arduino & pigpio
  • Raspberry Pi 4
