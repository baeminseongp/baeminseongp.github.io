Tic‑Tac‑Toe Game Robot Arm

Educational robot that plays O & X using vision and manipulation.

An educational project where a robot arm recognizes O and X tokens using a CNN and places them on the board. Demonstrates integration of deep learning, vision, and precise arm control.

Key Features

  • O / X recognition via Pytorch& OpenCV
  • Game logic & decision‐making
  • Real‐time image capture & processing

Tech Stack

  • Python, Pytorch
  • OpenCV
  • ROS2
