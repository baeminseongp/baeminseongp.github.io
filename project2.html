Crowd‑Based Navigation & ETA Prediction

Congestion‑aware path planning and arrival time estimation.

This system builds a time‑dependent costmap from public crowd‑density datasets and uses it to compute safe, efficient routes. Speeds and distance‐bias parameters are adjusted dynamically based on real‐time congestion levels to generate precise ETA predictions.

Key Features

  • Ingestion of time‐series crowd density data
  • Costmap generation reflecting congestion risk
  • Dynamic speed & distance bias tuning
  • Sigmoid smoothing for stable ETA outputs

Tech Stack

  • ROS2 (Nav2, Costmap Plugins)
  • Python (NumPy, JSON handling), C++
  • SpatioTemporalVoxelLayer for obstacle avoidance
  • Public crowd‐density APIs
