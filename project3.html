Crash Lab: Hospital Guidance Robot

GPT‑3.5 Turbo API + LiDAR‑based navigation.

Developed a hospital‐guide robot that converses with patients via the GPT‑3.5 Turbo API and navigates hospital corridors using LiDAR‐based SLAM and ROS2 Nav2 for localization and path planning.

Key Features

  • Natural language Q&A powered by OpenAI GPT‑3.5 Turbo
  • LiDAR‐based mapping & AMCL localization
  • Dynamic path planning with Nav2
  • Text‑to‑speech responses for interactions

Tech Stack

  • ROS2 (Nav2, rclcpp)
  • YDLidar X2 & SpatioTemporalVoxelLayer
  • OpenAI GPT‑3.5 Turbo API
  • Python, WebSocket communication
