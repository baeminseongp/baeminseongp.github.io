NUVI Autonomous Mobility

Autonomous robotics project utilizing ROS and AI technologies.

NUVI is an autonomous mobility system designed for airport environments. It integrates real-time crowd density analysis for dynamic path planning. It can be controlled by web UI.

Key Features

  • Crowd‐aware costmap generation from time‐dependent density data
  • ROS2 Nav2 integration with RealSense D455 pointcloud

Tech Stack

  • ROS2 (Nav2 : AMCL, DWB)
  • Intel RealSense D455
  • RTAB-Map & ORB-SLAM3
  • Python, C++
