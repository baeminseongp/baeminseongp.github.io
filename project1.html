NUVI Autonomous Mobility

Autonomous robotics project utilizing ROS and AI technologies.

NUVI is an autonomous mobility system designed for airport environments. It integrates real-time crowd density analysis for dynamic path planning, predicts ETA accurately, and provides intuitive visual feedback via projection UI.

Key Features

  • Crowd‐aware costmap generation from time‐dependent density data
  • Dynamic ETA prediction with sigmoid‐based speed adjustment
  • Projection UI for pedestrian–robot interaction
  • ROS2 Nav2 integration with RealSense D455 pointcloud

Tech Stack

  • ROS2 (Nav2, AMCL, DWB)
  • Intel RealSense D455
  • RTAB-Map & ORB-SLAM3
  • Python, C++
